# TaskBot
[robot mascot image](https://raw.githubusercontent.com/roden0/todo-cli/main/img/robot.svg)
Todo task list CLI Node applicaiton.

## Setup

install dependencies
> npm i

Run the application
> node app

## Available options:
|Command         |Description                    |
|----------------|-------------------------------|
|Create task    |creates a new task (description text is mandatory|
|List tasks     |shows the object of tasks|
|Exit           |closes the application|
